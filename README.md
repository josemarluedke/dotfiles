# Dotfiles to setup your bash environment.

## Usage

Clone repository:

```$ git clone git://github.com/josemarluedke/dotfiles.git ~/.dotfiles```

Install the dotfiles:

```$ sh ~/.dotfiles/install.sh```

Restart your terminal and you are ready to go!
